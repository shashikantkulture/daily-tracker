# 📋 ClientFlow Pro — SaaS Task Tracker

A high-performance, premium SaaS dashboard for professional task management and client collaboration.

## 🚀 Key Features
- **🎨 Premium UI:** Custom Dark Red (Admin) and Navy Blue (Client) themes with glassmorphic effects.
- **⚡ Real-time Sync:** Powered by **Supabase** for instant updates across all devices.
- **🤝 Presence Tracking:** See exactly when clients are live on their portal.
- **⏱️ Time Management:** Integrated work session logging with history tracking.
- **📱 Mobile Optimized:** Native touch-swipe gestures for administrative control on the go.
- **📍 Floating Assigner:** Draggable (Right-click) task assignment widget for extreme efficiency.
- **📅 Interactive Timelines:** Real-time PC timelines for both daily routines and task overviews.

## 🏗️ Tech Stack
- **Frontend:** Vanilla HTML5, CSS3 (Premium Glassmorphism), ES6+ Javascript.
- **Backend/DB:** **Supabase** (PostgreSQL + Real-time engine).
- **Hosting:** **Vercel** (Automatic GitHub Deployments).

## 🛠️ Usage (Admin)
- **Arrows/Buttons:** Navigate between 7+ active client columns.
- **Right-Click Drag:** Move the **Assign Task** widget anywhere on your screen.
- **Hover:** Routine and Task timelines expand on hover for detailed viewing.
- **Sync:** Auto-syncs everything; no manual saving required.

## 🔗 Deployment
Previously hosted on Netlify, now migrated to **Vercel** for enhanced privacy and enterprise-grade performance.
