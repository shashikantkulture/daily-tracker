# 📋 Daily Client Work Tracker — Shashikant

A fast, clean, app-like productivity dashboard for daily client work tracking.

## Features
- 🎨 Dark Red premium theme
- 🗂 Slide-based client navigation (7 clients)
- ✅ Task toggle with green completion state
- ✏️ Inline edit client names & tasks
- ➕ Add / 🗑 Delete tasks per client
- ⏱ 40-minute countdown timer per client
- 📋 Backlog notes panel with C1–C7 client tabs
- 💾 All data persisted in `localStorage`
- 📱 Fully responsive (mobile + desktop)

## Deployment
Hosted on **GitHub Pages** — works on any device via shared link.

## Usage
- Use `← →` arrow keys or **Prev / Next** buttons to navigate between clients
- Click a task to mark complete (turns green)
- Hover a task to see **Edit ✏️** and **Delete 🗑** icons
- Double-click the client name to rename it
- Click **📋 Complete Backlog** to open the notes panel
- Timer auto-stops and logs time spent per client

## Tech Stack
- Pure HTML + CSS + JavaScript (single file)
- No backend, no database
- `localStorage` for all persistence
